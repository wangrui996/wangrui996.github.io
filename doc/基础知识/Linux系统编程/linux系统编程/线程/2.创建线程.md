# 2.创建线程  


## pthread_self 函数  

* 功能: 获取线程ID   对应于进程中的getpid()函数   
    * 线程ID用于在进程内部，线程的标识(两个进程间线程ID允许相同)  
    * 使用 ps -Lf 进程号  查看某一进程下的LWP变量表示的是进程号，用于CPU调度。
