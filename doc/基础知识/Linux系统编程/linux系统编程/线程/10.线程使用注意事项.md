# 10.线程使用注意事项  

* malloc和mmap申请的内存在线程间是共享的，因此可以被其他线程释放  
* 第4条,在某个线程中调用fork()函数,那它创建出来的子进程,只含有调用它的这一个线程  一般不这么用,如果确实需要可以在pthread_create之前就fork
