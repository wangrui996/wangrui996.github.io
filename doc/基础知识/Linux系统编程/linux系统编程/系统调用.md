# 系统调用  

系统调用：操作系统实现并提供给外部应用程序的编程接口(API),是应用程序同系统之间数据交互的桥梁

以下图为例：  

* 真正意义上的系统调用应该是 sys_write 函数  但操作系统又做了一层浅封装向上提供了write函数，平时一般把write、open这些也叫作系统调用但严谨的说应该是系统函数  

* 用户程序等通常在0-3G位置，3-4G位置为内核
