# 9.中断系统调用  

* 系统调用分为**慢速系统调用**和**其他系统调用**

* 慢速系统调用  
    * 可能会使进程永久阻塞的一类。如果在阻塞期间收到一个信号，该系统调用就被中断，不再继续执行(早期)，也可以设定系统调用是否重启。
    * 如 read、write、pause、wait(回收子进程，但是子进程一直不死，就可以阻塞)  
* 其他系统调用
    * getpid、fork...


* 慢速系统调用被信号中断后可以重启
