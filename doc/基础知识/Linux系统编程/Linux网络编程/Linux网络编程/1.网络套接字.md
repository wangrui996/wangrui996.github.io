# 1.网络套接字  


## 网络套结字  socket  

* socket英文翻译是插座, 把…装入插座

* 通信过程中，套接字一定是成对存在的 (通信过程中)  
    * 只有一个可以但无法实现网络通信
    * 进程间通信的管道, mmap映射区都是只有一个

## Linux内核实现套接字的原理  

* 一个虚方框表示一个套接字，在一个套接字上只有一个文件描述符
    * 管道有读端和写端两个文件描述符 一个缓冲区  
    * 套接字只有一个文件描述符，套接字内部实现有两个缓冲区,一个发送缓冲区,一个接受端缓冲区
    * 因此数据的流动： 从其中一个套接字的文件描述符写，另一个文件描述符读
