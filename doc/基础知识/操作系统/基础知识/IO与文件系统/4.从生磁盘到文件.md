# 4.从生磁盘到文件  


## 操作系统对磁盘使用的第三层抽象  

* **明确这次抽象的目的：希望从文件 得到 盘块号**  
    * 用户眼中的文件是字符序列，字符流 因此，任务是 如何根据字符流计算出它属于哪一个盘块  
* **核心： 建立字符流到盘块集合的映射关系**
